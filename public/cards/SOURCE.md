# Playing card assets

Card faces imported from Webisso Playing Cards.

- Source: https://webisso.github.io/playing-cards/
- Repository: https://github.com/webisso/playing-cards
- License: MIT

Imported files are stored with this app's existing card ids, such as `spades-A.svg`.

Court cards (`J`, `Q`, `K`) were then replaced with classic court-card artwork
rendered from `@letele/playing-cards`, based on Adrian Kennard's CC0 public
domain SVG playing card designs.

- Package: https://www.npmjs.com/package/@letele/playing-cards
- Source design: https://www.me.uk/cards/
- License: CC0-1.0
